Pada website NoBox.Ai memiliki beberapa tampilan umum yaitu:

## **Dasbor**

Berisikan ringkasan informasi aktivitas secara real time dan dapat mempermudah Anda untuk mengambil keputusan.

<ImageZoom
  src="/images/id/dasbor%20kirim%20pesan.png"
  alt="Tampilan Dasbor NoBox"
  width={800}
  height={450}
  className="rounded-lg border shadow-sm mb-4"
/>

Dalam tampilan dasbor terdapat fitur umum yang punya fungsi masing-masing:

| Fitur / Tombol   | Fungsi                                                    |
| :--------------- | :-------------------------------------------------------- |
| **Filter Tahun** | Filter data dasbor berdasarkan periode per tahun          |
| **Auto Refresh** | Merefresh data di dasbor secara otomatis setiap 10 detik  |
| **Bantuan**      | Menampilkan bantuan atau _knowledge base_                 |
| **Full Screen**  | Menampilkan data grafik secara layar penuh (_fullscreen_) |

---

## **Kanban**

Salah satu metode dalam manajemen proyek dengan cara membuat visualisasi menggunakan papan, kolom, dan kartu untuk mengelola tugas dan alur kerja secara efektif.

<ImageZoom
  src="/images/id/PenawaranTampilan.PNG"
  alt="Tampilan Kanban NoBox"
  width={800}
  height={450}
  className="rounded-lg border shadow-sm mb-4"
/>

Dalam tampilan kanban terdapat fitur umum dengan fungsi sebagai berikut:

| Fitur / Tombol | Fungsi                                                                  |
| :------------- | :---------------------------------------------------------------------- |
| **Papan**      | Memilih data projek yang telah dibuat yang akan ditampilkan pada kanban |
| **Filter**     | Filter data kanban berdasarkan filter data yang tersedia                |
| **Cari**       | Pencarian data sesuai dengan kata kunci                                 |
| **Kanban**     | Menampilkan spesifikasi alur kerja secara visual                        |

---

## **List**

Tampilan yang berisi suatu daftar data yang sudah dibuat pada menu-menu tertentu.

<ImageZoom
  src="/images/id/TampilanKirimPesan.PNG"
  alt="Tampilan List NoBox"
  width={800}
  height={450}
  className="rounded-lg border shadow-sm mb-4"
/>

Fitur umum pada tampilan list:

| Fitur / Tombol      | Fungsi                                                     |
| :------------------ | :--------------------------------------------------------- |
| **Cari**            | Pencarian data sesuai dengan kata kunci                    |
| **Refresh**         | Melakukan penyegaran data secara manual                    |
| **Column Picker**   | Menampilkan atau menyembunyikan kolom pada tabel (grid)    |
| **Restore Default** | Mengembalikan tampilan kolom sesuai dengan pengaturan awal |
| **Export**          | Mengunduh data dalam format **Excel** atau **PDF**         |
| **Auto Refresh**    | Merefresh data secara otomatis setiap 10 detik             |
| **Filter Status**   | Filter data sesuai dengan kategori atau status tertentu    |

---

## **Formulir**

Tampilan yang berfungsi untuk menambahkan data baru atau untuk mengedit data yang sudah ada.

<ImageZoom
  src="/images/id/PengaturanFormulirTampilan.PNG"
  alt="Tampilan Formulir NoBox"
  width={800}
  height={450}
  className="rounded-lg border shadow-sm mb-4"
/>

Dalam tampilan form terdapat fitur umum sebagai berikut:

| Fitur / Tombol      | Fungsi                                                         |
| :------------------ | :------------------------------------------------------------- |
| **Grup / Kategori** | Mengelompokkan beberapa isian sesuai bagiannya agar lebih rapi |
| **Simpan**          | Menyimpan data baru atau perubahan data yang sudah diedit      |
| **Hapus**           | Menghapus data yang dipilih dari sistem                        |
