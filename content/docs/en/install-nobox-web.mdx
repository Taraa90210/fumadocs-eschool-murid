If the installation is successful, two folders will appear in the selected installation directory: **NoboxChatManager** and **NoboxChatSession**.

## **Nobox Chat Manager**

Used to integrate web chat, social media, and instant messaging into automation accounts, making it easier to manage customer interactions and business prospects.  
To run the **NoBox Chat Manager** application, follow these steps:

- First, open the folder where you installed the NoBox application. This folder is usually located in the directory you selected during installation
- Select the **NoboxChatManager** folder, open the **[Bin]** folder, and run the **NoboxChatManager.exe** file
