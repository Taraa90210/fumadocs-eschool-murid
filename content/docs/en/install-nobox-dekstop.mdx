  [NoBox Release](https://bit.ly/mynobox-release)
- After the download is complete, extract the folder
- Open the **NoboxDesktop ➔ Bin ➔ locate the NoBox Desktop application**
- Double-click to launch the application

## **NoBox Desktop Login**

To open NoBox Desktop, log in using the account you previously created on the website  
[id.nobox.ai](https://id.nobox.ai/).

<ImageZoom
  src="/images/id/Login%20Dekstop.png"
  alt="NoBox Desktop Login Interface"
  width={800}
  height={450}
  className="rounded-lg border shadow-sm mb-4"
/>

- Open the NoBox Desktop application
- Enter the username and password from your NoBox.AI web account
- If you do not yet have an account, please follow the guide in  
  [Registration](registrasi.md)
- Click **Login**, and NoBox Desktop will start successfully

**Note:** NoBox Desktop can only be run on one computer at a time.
