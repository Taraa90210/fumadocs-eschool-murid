The NoBox.AI website provides several common interface views, including:

## **Dashboard**

Contains a real-time summary of activity information and helps users make decisions more easily.

<ImageZoom
  src="/images/id/dasbor%20kirim%20pesan.png"
  alt="NoBox Dashboard Interface"
  width={800}
  height={450}
  className="rounded-lg border shadow-sm mb-4"
/>

The dashboard view includes general features with the following functions:

| Feature / Button | Function                                                |
| :--------------- | :------------------------------------------------------ |
| **Year Filter**  | Filters dashboard data based on yearly periods          |
| **Auto Refresh** | Automatically refreshes dashboard data every 10 seconds |
| **Help**         | Displays help or the _knowledge base_                   |
| **Full Screen**  | Displays chart data in full-screen mode (_fullscreen_)  |

---

## **Kanban**

A project management method that visualizes work using boards, columns, and cards to manage tasks and workflows effectively.

<ImageZoom
  src="/images/id/PenawaranTampilan.PNG"
  alt="NoBox Kanban Interface"
  width={800}
  height={450}
  className="rounded-lg border shadow-sm mb-4"
/>

The kanban view includes the following general features:

| Feature / Button | Function                                                        |
| :--------------- | :-------------------------------------------------------------- |
| **Board**        | Selects an existing project to be displayed on the kanban board |
| **Filter**       | Filters kanban data based on available filter options           |
| **Search**       | Searches data based on keywords                                 |
| **Kanban**       | Displays workflow stages visually                               |

---

## **List**

A view that displays a list of data created within specific menus.

<ImageZoom
  src="/images/id/TampilanKirimPesan.PNG"
  alt="NoBox List Interface"
  width={800}
  height={450}
  className="rounded-lg border shadow-sm mb-4"
/>

Common features in the list view include:

| Feature / Button    | Function                                              |
| :------------------ | :---------------------------------------------------- |
| **Search**          | Searches data based on keywords                       |
| **Refresh**         | Manually refreshes the data                           |
| **Column Picker**   | Shows or hides table columns (grid)                   |
| **Restore Default** | Restores column layout to the default settings        |
| **Export**          | Downloads data in **Excel** or **PDF** format         |
| **Auto Refresh**    | Automatically refreshes data every 10 seconds         |
| **Status Filter**   | Filters data based on specific categories or statuses |

---

## **Form**

A view used to add new data or edit existing data.

<ImageZoom
  src="/images/id/PengaturanFormulirTampilan.PNG"
  alt="NoBox Form Interface"
  width={800}
  height={450}
  className="rounded-lg border shadow-sm mb-4"
/>

The form view includes the following general features:

| Feature / Button     | Function                                               |
| :------------------- | :----------------------------------------------------- |
| **Group / Category** | Groups input fields by section for better organization |
| **Save**             | Saves new data or updates existing data                |
| **Delete**           | Deletes the selected data from the system              |
