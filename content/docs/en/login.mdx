Select the **[Subscription]** menu, then download the installer that matches your computerâ€™s operating system.

Run the downloaded NoBox installer. Select the server you want to use, then enter the username and password associated with your registered NoBox account.

If the installation is successful, two folders will appear in the selected installation location: **NoboxChatManager** and **NoboxChatSession**.
