The Extracurricular Feature allows you to view all extracurricular activities available at school, register for activities you're interested in, and see the list of extracurriculars you've already joined.

### 1. View All Extracurriculars

You can see a complete list of all extracurricular activities available at school. Each extracurricular is displayed in a card format containing:

- **Extracurricular Name** - Activity title
- **Instructor Name** - Teacher or instructor who guides
- **Description** - Brief explanation about the activity
- **Availability Status** - Whether there are still spots available or it's full

### 2. View Joined Extracurriculars

You can see the list of extracurriculars you've already joined by switching to the "My Extracurriculars" tab. Here you will see:

- All approved extracurriculars
- Your registration status
- Join date
- Instructor information and activity description

### 3. Search for Extracurriculars

Use the search feature to quickly find the extracurricular you're looking for:

1. Tap the search field at the top
2. Type the name of the extracurricular you want to find
3. Search results will appear automatically
4. Tap the ‚ùå button to clear the search

### 4. Register for Extracurriculars

To register for a new extracurricular:

1. Select the extracurricular you're interested in
2. Make sure the status shows "Available" (not "Full")
3. Tap the green **"Join"** button
4. Wait for registration confirmation
5. Status will change to **"Pending Approval"**
