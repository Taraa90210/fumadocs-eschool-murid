### Gallery Page

The **Gallery** page in eSchool Mobile displays documentation of school activities in photo or video format, stored by category and academic year.

Key features on this page:

1. **Gallery Search**\
   Use the search field to find activity documentation by name or keyword.
2. **Academic Year Filter**\
   Select a specific academic year to display photo galleries from that period.
3. **Activity Album List**\
   Each activity is displayed in card format with:
   * **Activity title** (e.g.: _PASKIB_, _PASKADEMA_)
   * Thumbnail of the first photo
   * Number of photos in the album (if more than one)
4. **Album Detail Navigation**\
   Tap an album to see all photos inside in more detail.
