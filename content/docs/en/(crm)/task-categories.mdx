## **Add Task Category**

To add a new task category, please open the **[CRM]** module, then click **[Tasks] ➔ [Categories] ➔ [Add Task Category]**. The **[Add Task Category]** dialog will appear.

<ImageZoom
  src="/images/id/Tampilan%20Tambah%20kategori%20Tugas.png"
  alt="NoBox Dashboard View"
  width={800}
  height={450}
  className="rounded-lg border shadow-sm mb-4"
/>

Complete the required fields, for example as shown below:

| Field | Description   | Example |
| ----- | ------------- | ------- |
| Name  | Category Name | VIP     |

After completing the required fields, click **[Save]** to store the newly created task category.

## **Edit and Delete Task Categories**

To edit a task category, first select the category you want to edit. Make the necessary changes and click **[Save]** to apply the updates.

To delete a task category, select the category you want to remove and click **[Delete]**. A confirmation dialog will appear asking whether you want to delete the data. Click **[Yes]** to proceed or **[No]** to cancel.

## **How to Use Task Categories**

1\. Open the **[Tasks]** menu, then select an existing task or add a new task.

2\. Click the category field, and the list of previously created categories will appear.

<ImageZoom
  src="/images/id/Tampilan%20pilih%20kategori.png"

3\. Select one of the categories and click **[Save]** to update the task data.
