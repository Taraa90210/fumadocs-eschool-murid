## **Edit and Delete Quick Reply Data**

To edit a quick reply, first select the data you want to edit. Make the necessary changes and click **[Save]** to save them.  
To delete a quick reply, select the quick reply data you want to remove, then click **[Delete]**. A confirmation dialog will appear asking whether you want to delete the data. Click **[Yes]** to proceed or **[No]** to cancel.

## **How to Use Quick Replies**

To send a quick reply, follow these steps:

1\. Select the **[Messages] ➔ [Inbox]** module to view all incoming conversations, then choose one of the messages.

2\. In the message input field, press **"/"** on your keyboard.
