- Conversation Archive is a feature used to store or sort messages that are no longer relevant or do not require further responses. By archiving, users can clean up their main conversation list without permanently deleting the messages. Click the **[![](/images/id/Arsip.png)]** icon to view archived conversations.
- You can filter conversations by Status, Channel, Account, Contact, and more. To filter conversations, click the **[![](/images/id/filter.png)]** icon. The filter panel will appear. Select the desired filters and click **[Show]** to apply them. Click **[Reset]** to clear the filters.

![](/images/id/filter.png)

- The Create Conversation feature allows users to start a new conversation with an existing contact. This is useful for initiating chats without manually searching for contacts or groups. Click the **[![](/images/id/percakapan%20baru.png)]** icon to create a new conversation.
- The Search feature is used to find conversations or contacts based on typed keywords.

![](/images/id/Pencarian.png)

## **Assigning Incoming Messages to Agents**

On the Inbox page, supervisors can allocate customers to available agents. To assign an agent, click the **[+ Agent]** button. A list of available agents will appear—click an agent’s name to select them.

## **How to Enable Auto Complete or Auto Archive**

To enable the Auto Complete and Auto Archive features, follow these steps:

1. Open the **[Settings] ➔ [Business Profile]** menu.
2. Enable Auto Complete or Auto Archive in the corresponding category.
3. Set the number of hours after which conversations will be automatically completed or archived.

<ImageZoom
  src="/images/id/Arsip%20Otomatis.png"
  alt="Auto Archive Settings"

## **Auto Complete**

On the Inbox page, Auto Complete is used to enable or disable automatic completion of conversations. This feature applies to conversations with the **In Progress** status.

<ImageZoom
  src="/images/id/Selesai%20Otomatis.png"
  alt="Auto Complete Feature"
  width={800}
  height={450}
  className="rounded-lg border shadow-sm mb-4"
/>

If you want to mark a conversation as completed immediately without waiting for Auto Complete, click the **[![](/images/id/titik%20tiga.png)]** icon and select **[Mark as Completed]**.

## **Auto Archive**

On the Inbox page, Auto Archive is used to enable or disable automatic archiving of conversations. This feature applies to conversations with the **Completed** status.

<ImageZoom
  src="/images/id/Kotak%20Masuk.png"
  alt="Auto Archive Feature"
  width={800}
  height={450}
  className="rounded-lg border shadow-sm mb-4"
/>

If you want to archive a conversation immediately without waiting for Auto Archive, click the **[![](/images/id/Arsip.png)]** icon to archive the conversation directly.

## **Conversation Information**

On the Inbox page, you can view conversation details by clicking the **[![](/images/id/information%20icon.png)]** icon.

<ImageZoom
  src="/images/id/Informasi%20Percakapan.png"
  alt="Conversation Information"
  width={800}
  height={450}
  className="rounded-lg border shadow-sm mb-4"
/>

There are several sections, each with its own function:

- Information (name, profile photo, and phone number).
- Conversation History is used to display completed conversation history.

![](/images/id/Riwayat%20Percakapan.png)

- Contact name and address information. To edit contact data, click the **[![](/images/id/edit%20contact.png)]** icon to be redirected to the contact edit page.

![](/images/id/Kontak%20Percakapan.png)

- Conversation information including the handling account, conversation creation date, and last message date.

![](/images/id/Percakapan.png)

- Notes are related to the selected conversation. Notes can be added, edited, and deleted so other agents can also contribute.

![](/images/id/Catatan%20Percakapan.png)

- Tags are used to group conversations and allow adding more than one tag to a single chat.

![](/images/id/Tag%20Percakapan.png)

- Promotions display promotional information linked to the conversation.

![](/images/id/Promosi%20Percakapan.png)

- Funnel is used to categorize marketing and support process stages, or to create a new funnel.

![](/images/id/Funnel.png)

- Deal is used to associate a deal with the conversation.

![](/images/id/Penawaran%20Percakapan.png)

- Agent displays information about which agents are handling the conversation.

![](/images/id/Agen%20Percakapan.png)

You can reply directly to incoming messages using NoBox. Open a conversation and respond according to the customer’s inquiry.

## **Delete Message**

On the Inbox page, the Delete Message feature is used to remove messages from a conversation. This only deletes messages within NoBox. To delete a message, select the message, click **[Delete]**, and confirm by clicking **[Yes]** or cancel by clicking **[No]**.

## **Quick Reply**

Quick Reply is a NoBox feature used as a message template to respond faster using text or media such as images, GIFs, and videos. To use Quick Reply, follow these steps:

1. In the message input field, press **"/"**.
2. A list of available quick replies will appear. Select one.
3. The quick reply will automatically appear in the message field. Click the **[![](/images/id/send.png)]** button to send it.
