## **Add Funnel**

To create funnel data, please click the **[Funnel]** menu **âž” [Add Funnel]**.  
The **[Add Funnel]** dialog will appear, then enter the funnel name in the dialog.

After you have entered the funnel name in the dialog above, click **[Save]** to store the data.

## **Edit and Delete Funnel Data**

To edit a funnel data, please select the data you want to edit. Make your changes and click **[Save]** to save them.  
To delete a funnel data, select the funnel data you want to delete, then click **[Delete]**. A confirmation dialog will appear asking whether you want to delete the data. Click **[Yes]** to proceed or **[No]** to cancel.

## **How to Use Funnel**

1\. Open the **[Inbox]** menu, then select one of the available conversations.

2\. Click the **[**![](/images/id/information%20icon.png)**]** icon to display the chat details.  
Click the Funnel section to view previously created funnel data or add a new funnel.

3\. Select one of the available funnel data.
